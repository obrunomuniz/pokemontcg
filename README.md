## :rocket: Run the project:
```shell
  # Clone this repository
  - git clone https://github.com/obrunomuniz/pokemontcg
  
  # Enter project directory
  - cd [project folder]
  
  # Install dependencies
  - yarn
  
  # Dependencies iOS
  - cd [project folder]/ios
  - pod install
  
  # Start android
  - yarn android
  
  # Start iOS
  - yarn ios
```
